Git

  • git init Расшифровка: Initialize repository.Назначение: Создать новый репозиторий Git в текущей папке.
  • git clone [url] Расшифровка: Clone repository. Назначение: Скопировать удалённый репозиторий на компьютер.
  • git add [file] Расшифровка: Add files to staging. Назначение: Добавить файлы в индекс (подготовить к коммиту).
  • git commit -m "message" Расшифровка: Commit changes. Назначение: Сохранить изменения в истории репозитория.
  • git push Расшифровка: Push changes. Назначение: Отправить локальные изменения на удалённый сервер.
  • git pull Расшифровка: Pull changes. Назначение: Загрузить изменения с удалённого сервера.
  • git status Расшифровка: Show status. Назначение: Показать статус изменений файлов.
  • git log Расшифровка: Show log. Назначение: Показать историю коммитов
